# 🛒 Sky Store - E-Commerce Website

A modern, responsive, and visually appealing E-commerce website built using HTML, CSS, and JavaScript.
Designed with a premium glassmorphism UI, smooth animations, and interactive features.

---

## 🌐 Live Demo

👉 https://kumarsubodh12.github.io/ecommerce-site/

---

## ✨ Features

* 🛍️ Product listing with modern UI
* 🛒 Add to Cart functionality
* 🔢 Cart item counter
* 💾 Local Storage (cart data saved even after refresh)
* 📱 Fully responsive design
* 📌 Sticky Navbar
* ✨ Hover effects & animations
* 🎨 Glassmorphism UI (blur + transparency)

---

## 🎨 UI Highlights

* Sky blue & white gradient theme ☁️
* Glass-style product cards 💎
* Smooth hover & click animations 😎
* Clean and minimal layout

---

## 🛠️ Tech Stack

* HTML5 (Semantic structure)
* CSS3 (Flexbox, Grid, Animations)
* JavaScript (DOM manipulation, Local Storage)
